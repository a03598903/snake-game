# 霓虹贪吃蛇 | Neon Snake Game

一个现代化的霓虹风格贪吃蛇游戏，采用原生 JavaScript 开发，具有丰富的游戏特性和炫酷的视觉效果。

## 游戏演示

在浏览器中打开 `snake.html` 即可开始游戏！

## 游戏特性

### 核心玩法
- **经典贪吃蛇机制**：控制蛇吃食物，避免撞墙和撞到自己
- **流畅的游戏体验**：基于网格的移动系统，响应灵敏
- **霓虹视觉风格**：炫酷的霓虹配色和动画效果

### 难度系统
三种难度等级可供选择：
- **简单模式**：速度较慢，适合新手
- **普通模式**：标准速度，平衡挑战
- **困难模式**：高速移动，考验反应力

### 多样化食物系统
4种不同类型的食物，带来不同的游戏效果：
- **普通食物** 🟢：+10分，最常见
- **加速食物** ⚡：+20分，游戏加速5秒
- **减速食物** 🐌：+15分，游戏减速5秒
- **奖励食物** 💰：+50分，额外分数奖励

### 障碍物模式
- 可选的障碍物挑战模式
- 随机生成12-15个障碍物
- 增加游戏难度和策略性

### 音效系统
- 内置音效管理器，使用 Web Audio API
- 吃食物音效
- 特殊食物音效
- 游戏结束音效
- 背景音乐循环播放
- 可随时开关音效

### 游戏控制
支持多种操作方式：
- **方向键**：↑ ↓ ← → 控制蛇的移动
- **WASD键**：W/A/S/D 控制移动
- **空格键**：暂停/继续游戏
- **ESC键**：打开暂停菜单
- **R键**：快速重新开始

### 排行榜系统
- 本地存储最高分记录
- 保存前10名成绩
- 记录难度、障碍模式和日期
- 新纪录特殊提示

### 数据持久化
- 自动保存游戏设置（难度、音效等）
- 保存历史最高分
- 使用 localStorage 本地存储

## 技术栈

- **纯原生 JavaScript**：无框架依赖
- **HTML5**：现代化语义标签
- **CSS3**：霓虹效果、动画、渐变
- **Web Audio API**：动态音效生成
- **LocalStorage**：数据持久化

## 快速开始

### 在线运行

1. 克隆或下载本仓库
```bash
git clone https://github.com/a03598903/snake-game.git
cd snake-game
```

2. 直接在浏览器中打开 `snake.html`
```bash
# Windows
start snake.html

# macOS
open snake.html

# Linux
xdg-open snake.html
```

### 使用本地服务器（推荐）

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 游戏说明

### 如何游玩
1. 在主菜单选择难度和游戏模式
2. 点击"开始游戏"或按回车键
3. 使用方向键或WASD控制蛇的移动
4. 吃到食物增加分数和蛇的长度
5. 避免撞墙、撞到自己或障碍物

### 得分规则
- 普通食物：10分
- 加速食物：20分
- 减速食物：15分
- 奖励食物：50分

### 游戏技巧
- 注意特殊食物的效果时间
- 在障碍物模式下提前规划路线
- 利用减速食物在紧急时刻获得喘息
- 不要贪心追求高分，稳健生存更重要

## 开发说明

### 代码结构

#### 主要类
- **SoundManager**：音效管理系统
- **SnakeGame**：游戏主类，包含所有游戏逻辑

#### 核心常量
- `GRID_SIZE`：游戏网格大小 (25x25)
- `SPEEDS`：不同难度的速度设置
- `FOOD_TYPES`：食物类型配置
- `DIRECTIONS`：移动方向定义

### 自定义配置

可以在 `snake-script.js` 中修改以下参数：

```javascript
// 调整网格大小
const GRID_SIZE = 25;

// 调整速度
const SPEEDS = {
    easy: 200,    // 毫秒
    normal: 120,
    hard: 70
};

// 调整食物分数和出现概率
const FOOD_TYPES = {
    NORMAL: { score: 10, probability: 0.7 },
    // ...
};
```
